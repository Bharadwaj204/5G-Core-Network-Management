5G Core Management Prototype Presentation Generator
=================================================

This folder contains a Python script to automatically generate a PowerPoint presentation for your 5G Core Management Prototype project.

Files included:
- generate-presentation.py: Python script to create the PowerPoint presentation
- presentation-requirements.txt: Required Python packages
- powerpoint-vba-template.txt: Template VBA code (alternative approach)

How to use the Python script:
1. Install the required package:
   pip install -r presentation-requirements.txt

2. Run the script:
   python generate-presentation.py

3. The presentation will be saved as "5G-Core-Management-Presentation.pptx" in the current directory

The generated presentation includes:
- 18 professionally formatted slides
- All the content from your presentation outline
- Consistent styling and formatting
- Proper slide layouts for titles and content

Note: You may want to customize the content, add your name and date to the title slide, and add any specific diagrams or images you want to include.

For manual PowerPoint creation:
- Use the powerpoint-vba-template.txt as a reference for VBA automation
- Follow the slide structure outlined in the presentation-script.md file
